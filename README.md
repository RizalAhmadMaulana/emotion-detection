# Emotion Detection Web Application

Aplikasi web Flask berbasis AI untuk mendeteksi emosi (anger, disgust, fear, joy, sadness) dari teks menggunakan pustaka Watson NLP.

## Fitur
* Analisis emosi teks secara real-time.
* Menampilkan skor tiap emosi dan emosi dominan.
* Penanganan error untuk input kosong.
